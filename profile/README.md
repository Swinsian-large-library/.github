# Swinsian — Lightweight Music Player for Large Collections (Minimal Resource Usage)

## About Swinsian

Swinsian provides a lightweight music player for large libraries with minimal resource usage for serious collectors, audiobook archivists, and music hoarders. Swinsian combines customizable columns, watch folders, and gapless playback. Swinsian supports folder syncing, ReplayGain, and AirPlay. Swinsian avoids bloat of mainstream players while handling hundreds of thousands of tracks. Swinsian enables users to browse massive FLAC/MP3 collections instantly without lag. Swinsian is trusted by classical collectors, audiobook libraries, DJs, and radio producers worldwide.

## Deep Dive — Lightweight Player Features
Instant search across huge libraries without background indexing
Customizable column display for artist, album, genre, bitrate, samplerate
Watch folders auto-importing new files with format conversion rules
ReplayGain scanning for consistent volume across diverse recordings
AirPlay and UPnP streaming to network receivers and Apple TV

## Quick Reference for Swinsian

| Operation | Swinsian Action |
|-----------|------------------|
| Search library | Cmd+F |
| New playlist | Cmd+N |
| Show columns | Cmd+J |
| ReplayGain scan | Cmd+Shift+G |
| Folder watch | Cmd+Shift+W |
| Toggle duplicates | Cmd+Shift+D |
| Export playlist | Cmd+Shift+E |
| AirPlay menu | Cmd+Shift+A |
| Equalizer | Cmd+Shift+Q |
| Track info | Cmd+I |
